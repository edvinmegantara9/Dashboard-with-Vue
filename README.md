
## Description

SILAPER VUE ADMIN

BAPPEDA KOTA PAGAR ALAM - SUMATERA SELATAN


### Installation
# install app's dependencies
```
$ npm install
```

#### Usage

``` bash
# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build

# run linter
npm run lint

# run unit tests
npm run test:unit

# run e2e tests
npm run test:e2e

```
